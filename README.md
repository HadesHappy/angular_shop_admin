# Angular ADMIN
Angular 4 admin dashboard using Django 2, Django REST framework and Python 3


#### Server Setup


    $ cd server
###### (Optional)
    $ virtualenv -p python3 venv
    $ source venv/bin/activate

#
    $ pip install -r requirments.txt
    $ ./manage.py runserver


#### Client Setup
    $ cd client
    $ npm install
    $ ng serve --open
